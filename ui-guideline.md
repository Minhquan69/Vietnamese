# UI Guideline - AI Vietnamese Learning Platform
- Poppins
- Nunito

## Heading Style
- bold
- clean
- modern
- large spacing

--------------------------------------------------
# COMPONENT DESIGN
--------------------------------------------------

## Cards
Requirements:
- rounded-xl or rounded-2xl
- soft shadow
- hover animation
- subtle gradient
- padding spacing

## Buttons
Requirements:
- rounded-full
- gradient backgrounds
- hover scaling
- transition effects

## Inputs
Requirements:
- large spacing
- soft border
- focus glow
- animated label

--------------------------------------------------
# DASHBOARD UI
--------------------------------------------------

Dashboard must include:
- XP widget
- streak widget
- continue learning card
- learning chart
- recommendation cards
- daily challenge section

--------------------------------------------------
# LESSON UI
--------------------------------------------------

Must feel immersive.

Include:
- sticky progress bar
- animated vocabulary cards
- audio player
- interactive quiz section
- AI speaking feedback panel

--------------------------------------------------
# QUIZ UI
--------------------------------------------------

Must feel interactive.

Include:
- answer animation
- progress indicator
- timer animation
- correct/wrong feedback
- reward animation

--------------------------------------------------
# VIDEO LEARNING UI
--------------------------------------------------

Include:
- synced subtitles
- clickable words
- transcript sidebar
- popup vocabulary panel
- save word button
