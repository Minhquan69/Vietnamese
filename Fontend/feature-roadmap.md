# Feature Roadmap

## Product Vision

Build a friendly, intelligent Vietnamese learning platform for foreigners from beginner to advanced level. The product should combine structured lessons, interactive practice, pronunciation support, cultural context, and AI-personalized guidance.

## MVP Completion

Current MVP already has:

- Account registration and login.
- Role-based user/admin navigation.
- Level, course, unit management.
- Video import and transcript search.
- Quiz and placement test foundation.
- User progress foundation.

Missing MVP pieces:

- Route guards.
- Learner dashboard.
- Lesson detail page with clear learning flow.
- Quiz explanations and review mode.
- Placement result -> recommended level.
- Consistent admin content workflow.
- Mobile-first navigation.

## Learner Features

1. Onboarding:
   - Native language selection.
   - Vietnamese goal selection: travel, work, study, family, culture.
   - Placement test or beginner start.
   - Recommended learning plan.

2. Daily learning:
   - Daily goal.
   - Lesson queue.
   - Review queue.
   - Streak tracking.
   - Skill progress.

3. Learning modules:
   - Vietnamese alphabet and pronunciation.
   - Six tones training.
   - Vocabulary with image/audio examples.
   - Grammar lessons.
   - Listening drills.
   - Speaking drills.
   - Reading passages.
   - Writing correction.
   - Scenario conversations.

4. Practice:
   - Multiple-choice quiz.
   - Fill-in-the-blank.
   - Sentence ordering.
   - Dictation.
   - Flashcards.
   - Spaced repetition.
   - Pronunciation recording.

5. Motivation:
   - Streaks.
   - Badges.
   - Certificates.
   - Level completion.
   - Weekly goals.

## AI Features

- AI tutor chat.
- AI pronunciation feedback.
- AI speaking roleplay.
- AI writing correction.
- AI-generated examples.
- AI-generated quiz variants.
- AI learning recommendation.
- AI mistake analysis.

All AI-generated learning content should be reviewed or flagged before it becomes official curriculum content.

## Admin Features

- Content builder for levels/courses/units.
- Media library.
- Quiz builder with parts, passages, questions, answers, and explanations.
- Placement test builder.
- Learner analytics.
- Content publishing workflow.
- AI content review queue.
- User role and status management.

## Community Features

- Lesson comments.
- Teacher feedback.
- Study groups.
- Practice partner matching.
- Community challenges.
- Report/moderation workflow.

- Progress cards.

- Flashcards.
- Spaced repetition.
